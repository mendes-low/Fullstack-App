# Fullstack-App
## How to start project on Windows: 
```
git clone https://github.com/Mendes-rep/Fullstack-app

cd Fullstack-app

pip -r requirements.txt

python manage.py runserver
```

## For Linux and Mac OS
```
git clone https://github.com/Mendes-rep/Fullstack-app

cd Fullstack-app

pip3 -r requirements.txt

python manage.py runserver
```
